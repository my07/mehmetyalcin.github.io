Sedna 360

.NET Core, AngularJS

Turizm sektörü göz önüne alınarak geliştirilen genişletilebilir ve ölçeklenebilir bir uygulamadır. Crm dahil otel ihtiyaçlarına göre dizayn edilmiştir. Sql ve Nosql veritabanlarını kullanarak birden fazla API ve Client ile çalışır, Api uygulaması için .net core ve client için html ve angularjs kullanır. Sürüm kontrolü git ile VSTS üzerinden gerçekleştirilir ve VSTS'nin CI-CD özellikleri kullanılarak istemcilere de dağıtılır. AWS nin RDS, S3, EC2, SES, SNS servisleri bu projede kullanılmıştır.

Sedna Tour Sale

Android Native, Web Api

Rehber bölge seçimi yaparak, bölgedeki otelleri seçip müşteri listesini görebilir ve tura katılmak isteyen müşteriyi seçerek istenilen tura satış yapabilir, satılan biletlerin sedna acente programına online olarak işlenmesi gerçekleştirilir. Bu satış sırasında otel alınış saatleri ister rehber tarafından ister daha önce tanımlanmış saate göre sisteme kayıt olur. Ödeme tipi kredi kartı veya nakit olarak seçildiğinde tahsilat kaydı da buna göre yapılır. Aynı zamanda rehberlerin konumları sedna acente programında canlı olarak izlenebilir.

Sedna Mobile POS

Android Native, Web Service

Kolay menü erişimi ile ürün seçilebilmesi, adisyon/masa takibi, adisyonları tahsil etme, birleştirme, ayrıştırma gibi işlemleri yapabilir, bunların müşteri hesaplarına otomatik olarak işlenmesini sağlar. NFC kart sistemini kullanan işletmelerde müşteriyi bu kart üzerinden bulup satışı gerçekleştirir.
